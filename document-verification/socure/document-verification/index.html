Document Verification

documentVerification​

ID+ Document Verification enables you to confirm the authenticity of a wide range of government-issued identification cards. This includes Driver's Licenses, Passports, Visas, and other official documents

referenceId​

The Reference ID is an identifier unique to this transaction.

reasonCodes​

The reasonCodes array contains all reason codes that influence the decision.

documentType​

This object returns information about the examined identification. This object is not returned for image errors.

Type​

Document type. Values: Drivers License, Passport, Identification Cards.

Country​

Country of origin for the document.

State​

State of origin for the Driver License. Note, for Passports and non-US Driver Licenses, the returned value is null.

Decision​

This object contains decision information.

Name​

This object returns the decision model used. This is configured when the account is created.

Value​

This object returns the decision. It contains one of the following four values: accept, review, resubmit, reject.
