Persona

Data (Placed into “Wrong” or “Correct” Categories)​

Account Comparison​

Compare if ID details like name or birthdate are inconsistent.

Portrait Age​

Detects if the age listed on the ID is different than the age estimated from the face portrait.

Barcode​

Detect if there is a barcode on the ID.

Barcode Inconsistency​

Detect inconsistencies in the barcode on the ID.

Blue​

Detect if the submission is blurry.

Color​

Detects if the colors on the ID are different than the expected colors.

Compromised Submission​

Detects if the submission can be found from a publicly available source (e.g. the internet).

Allowed Country​

Detects if the country of the ID is not allowed based on template requirements.

Allowed ID type​

Detects if the ID type submitted is not allowed based on template requirements.

Electronic replica​

Detect if the submission is a replica of an image presented on another electronic device or screen.

Government Entity ID​

Detect if there is a government ID in the submission.

Expiration​

Detects if the ID is expired.

Extracted Properties​

Detects if required ID details like name or birthdate are extracted.

Extraction Inconsistencies​

Detect if the details extracted from the front of the ID are different than the details extracted from the barcode.

Glare​

Detects if the submission has glare.

Handwriting​

Detects if the handwriting is found on the ID.

Public Figure​

Detect if face portrait appears to match that of a known public figure

Repeat​

Detect if the ID details and face portrait match that of a previously submitted ID on a different account

Inconsistent Repeat​

Detect if there is any inconsistent repeats

Selfie-to-ID Comparison​

Detects if face portrait on the ID and the selfie appear to be different faces

Image Tampering​

Detect if image has been tampered with

Blur​

Detects if image is blurry

Compromised Submission​

Detect if submission

Double Side​

Detect if ID is double sided

MRZ​

Detect if ID is machine-readable

PO Box​

Detect if PO Box exists

Portrait Clarity​

Detects if face portrait is clear or blurry

Portrait​

Detects if portrait is real person

Processable Submission​

Detect if submission is processable

Valid Dates​

Detect if dates on ID are valid

Multiple Faces​

Detect if there is more than one face in selfie

Pose Position​

Detects if pose of portrait seems illegitimate

Suspicious Entity​

Detects if individual is suspicious

Address​

Detect if address is legitimate

Deliverable Address​

Detect if address is legitimate

Birthdate​

Detect if birth date matches birthdate on ID

Alive​

Detects if individual is alive or deceased

Name​

Detect if name matches ID

Phone Number​

Detecting if the phone number is real

SSN​

Detect if SSN is legitimate

Recency​

Detects if document submitted is old

Type​

Detect if type is expected or unexpected
